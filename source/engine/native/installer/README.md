# WebDriver runtime installers

The official WildTangent **WebDriver 4.1** runtime, as distributed in 2004.
© WildTangent / Gamigo — preservation copy, see `../../../../LEGAL.md`.

- `Install_Webdriver.exe` — interactive installer.
- `WebDriverSilentInstall.exe` — silent/unattended installer.

Installing this on Windows (or Wine) provides the native engine (`../dlls/`) that
the decompiled game in `../../../` runs on.
