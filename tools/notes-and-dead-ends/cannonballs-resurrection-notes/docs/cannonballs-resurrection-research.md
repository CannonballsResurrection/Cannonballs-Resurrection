# Research: the Cannonballs! Resurrection project & its author

Companion to `cannonballs-research.md` (which covers the game). This is the research dive into the
**fan project** and its creator. (Credits live in `resurrection-archive/CREDITS.md`; this is the research detail.)

## The author
- **"Ken" / Kenneth** — identified on his own Contact page. Aliases: **htennek98** ("Kenneth98"
  reversed), **CaptSmokey6**, **CrazyCannon**.
- Contact (as he published it): `resurrectcannonballs@gmail.com`; later `ken@captsmokey.com` /
  `captsmokey6@gmail.com`. MySpace `myspace.com/cannonballs_resurrection`.

## The project (c. 2008–2016)
- Sites: `resurrectcannonballs.googlepages.com` (2008) → `cannonballsresurrection.com` (2009) →
  files also at `sites.google.com/site/cannonballsresurrection`. Forums `cannonballs.mybb3.org`
  (not archived). A 2016 follow-up appears as a comment on the *WildTangent Restoration* blog (see
  `kenneth-2016-followup.md`).
- Goal: fully resurrect *Cannonballs!*, released as **freeware**.
- Distributed: the game, the required **Microsoft Java VM**, soundtrack, wallpaper, and his
  **all-islands patch** (`island.exe`), plus an "Everything Pack."
- The patch unlocks islands **one at a time** (close game → apply next island's patch → reopen),
  removes the in-launcher "Register Now" trial gate, and changes the exit popup. He was also working
  toward fully cracking the retail version (never finished; project went dormant).
- His grievance with WildTangent: they cited low lobby traffic + new-tech incompatibility + server
  cost; he countered it ran fine on then-current Windows and the lobby often had 50+ players.

## Why the game half-works today
- **Single-player: playable.** Independently confirmed by YouTuber *StellarGwynn*'s recorded runs
  (Tropicali, Moonlight Cove, Voodoo Isles) on real Windows.
- **Multiplayer (Guest Login / lobby): permanently dead** — WildTangent shut down the server.

## How to use his files (from his own 2016 instructions)
1. Install the **Microsoft Java VM** (`msjavx86`) — required for all WildTangent games.
2. Install the game (`cannonballs-setup.exe`).
3. Run his `island.exe` patch. On 64-bit Windows, **manually launch `cannonballs.exe`** from the
   install folder (the black-and-white-icon launcher); pick an island; Launch Game.

## Gaps
- The forums and his MySpace aren't archived; little else about "Ken / htennek98 / CrazyCannon /
