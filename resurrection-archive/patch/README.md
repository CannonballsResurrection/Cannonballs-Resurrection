# Kenneth's "Access all Islands" patch

`island.exe` is Kenneth's installer (built with Clickteam Install Creator). Run it on top of an
installed Cannonballs!; it adds the **"Cannonballs! Access all Islands"** launcher (`cannonballs.exe`)
which:
- removes the in-launcher **"Register Now"** trial gate, and
- lets you install each island's area patch (**Access Crossbones / Volcano / Pyramids / Nightbridge /
  Ziggurat / Voodoo Isles / Overboard / Moonlight Cove / Tropical / Skull Isle / Old Gods**), then
  **Launch Game**.

Per Kenneth: islands are accessed **one at a time** — close the game, apply the next island's patch,
reopen. This is entirely his work.
