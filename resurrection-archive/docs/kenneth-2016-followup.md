# Kenneth's 2016 follow-up — a sequel to the Cannonballs! Resurrection website

Years after the original Cannonballs! Resurrection site went dormant, **Kenneth resurfaced as
"CaptSmokey6"** and left a comment on the *WildTangent Restoration* blog with updated download links,
a crucial running tip, and current contact info. It's effectively the next chapter of his project, so
it's preserved here.

**Source:** WildTangent Restoration blog — "Here are the list of WildTangent games that…"
<https://wildtangentgames00.blogspot.com/2016/02/here-are-list-of-wildtangent-games-that.html>

## What this adds to the record
- **The definitive way to run the patched game:** install the game + Microsoft Java VM, run the patch,
  then **manually launch `cannonballs.exe`** (the black-and-white-icon launcher) from the install
  folder — especially on 64-bit Windows. This is the same `cannonballs.exe` ("Access all Islands"
  launcher) the patch installs; from it you pick an island and Launch Game.
- A **third hosting location** for his files surfaced: `sites.google.com/site/cannonballsresurrection`
  (after the earlier googlepages and cannonballsresurrection.com homes), with Wayback copies of
  `island.zip` and `msjavx86.zip`.
- **Microsoft Java VM is mandatory** for all WildTangent games — if the installer doesn't bundle it,
  install `msjavx86` first.
- Updated contact: **ken@captsmokey.com** (his own domain) and **captsmokey6@gmail.com**, with an open
  offer to help anyone using his patch — a decade after he first built it.

## Note on running it in 2026
Kenneth's instructions are exactly right and his patch still works as designed. The only remaining
hurdle isn't his — it's that the WLD3 (DirectX 7/8) engine needs real GPU/DirectX, which software
emulators (QEMU/UTM) on Apple Silicon don't provide. See our attempt notes (moved out of this archive):
[running-on-macos.md](../../tools/notes-and-dead-ends/cannonballs-resurrection-notes/docs/running-on-macos.md)
and [ATTEMPTS.md](../../tools/notes-and-dead-ends/cannonballs-resurrection-notes/ATTEMPTS.md).
